Migration
1. install dotnet ef
https://learn.microsoft.com/en-us/ef/core/cli/dotnet

2. create database
3. change database connection string in appsetting.json
4. go to project folder right click open with terminal
5. run command "dotnet ef update database"

OR Import SQL file

1. Create database "StudentEnrollmentSystemDB"
2. open and run the "script.sql" In MSSQL
